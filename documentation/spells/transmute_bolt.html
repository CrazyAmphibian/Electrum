EL_TRANSMUTEBOLT
Transmuting Bolt, when colliding with an entity that stores materials, will, for each material, change it into a different material. The new material is randomly picked from any material, as long as both materials share enough tags.
to determine this, the number of similar tags are taken, then, if that number is at least or more than 3/4ths the number of tags that the material with the least number of tags has (rounded up), it will be able to change into it.
